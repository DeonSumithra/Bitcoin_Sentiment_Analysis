# ₿ Bitcoin Market Sentiment vs Trader Performance Analysis

## 📌 Project Overview

This project explores the relationship between **Bitcoin market sentiment** and **trader profitability** using:

* 📊 Bitcoin Fear & Greed Index Dataset
* 📈 Hyperliquid Historical Trader Data

The goal is to uncover hidden trading patterns and understand how emotions like **Fear** and **Greed** influence market behavior and trader performance.

## 📂 Datasets Used

### 🧠 Bitcoin Fear & Greed Index Dataset

**Columns:**

* `date`
* `classification`

### 💹 Hyperliquid Historical Trader Dataset

**Columns Include:**

* `Timestamp IST`
* `Side`
* `Closed PnL`
* `Symbol`
* `Size`
* `Account`
* `Execution Price`
* `Leverage`

## 🔍 Key Insights

* 🚀 Extreme Greed periods showed the highest average profitability.
* 📈 Fear periods generated the highest total profits due to increased trading activity.
* 📉 SELL trades performed strongly during Greed conditions.
* 💰 BUY trades performed better during Fear markets.
* 🧠 Market sentiment significantly influenced trader behavior and profitability.
